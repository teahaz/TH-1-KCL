# Library contents


### Diagram symbols

The diagram symbols library contains a set of over simplified building blocks of the TH ecosystem. The purpose of these is purely to make visual diagrams and *not* for use in real schematics.


## TH-1-CC (common components)

This is a library of components that have been previously used in parts of TH. The primary purpose of this is to keep components consistent across TH projects. Additionally this lets me order certain components in bulk, reducing cost.
